# 现有第三方依赖

本文档列出了当前支持的第三方库及其详细信息。

## 库列表

### tslib - 触摸屏校准库

| 属性 | 值 |
|------|-----|
| **用途** | Qt 平台插件 linuxfb 的触摸屏校准支持 |
| **当前版本** | 1.22 |
| **下载地址** | https://github.com/libts/tslib/releases/download/1.22/tslib-1.22.tar.xz |
| **构建系统** | Autotools (./configure) |
| **Qt 特性** | FEATURE_tslib |
| **依赖关系** | 无外部依赖 |
| **输出文件** | libts.so, libts.so.0 |
| **头文件** | tslib/libts.h |
| **pkg-config** | tslib.pc |

**配置选项**：
```bash
TSLIB_ENABLED="${TSLIB_ENABLED:-true}"
TSLIB_BUILTIN_VERSION="${TSLIB_BUILTIN_VERSION:-1.22}"
```

**交叉编译配置**：
```bash
--host=arm-linux-gnueabihf
--prefix=/usr
--enable-shared=yes
--enable-static=no
```

### libsndfile - 音频文件读写库

| 属性 | 值 |
|------|-----|
| **用途** | 音频文件格式读写（WAV, FLAC, OGG 等），PulseAudio 的依赖 |
| **当前版本** | 1.2.2 |
| **下载地址** | https://github.com/libsndfile/libsndfile/releases/download/1.2.2/libsndfile-1.2.2.tar.xz |
| **构建系统** | Autotools (./configure) |
| **Qt 特性** | 间接支持（通过 PulseAudio） |
| **依赖关系** | 无外部依赖 |
| **输出文件** | libsndfile.so |
| **头文件** | sndfile.h |
| **pkg-config** | sndfile.pc |

**配置选项**：
```bash
SNDFILE_ENABLED="${SNDFILE_ENABLED:-true}"
SNDFILE_BUILTIN_VERSION="${SNDFILE_BUILTIN_VERSION:-1.2.2}"
```

**交叉编译配置**：
```bash
--host=arm-linux-gnueabihf
--prefix=/usr
--disable-static
--enable-shared
--disable-sqlite
--disable-external-libs
```

### PulseAudio - 音频系统

| 属性 | 值 |
|------|-----|
| **用途** | Linux 音频服务器，Qt Multimedia FFmpeg backend 在 Linux 上需要 |
| **当前版本** | 17.0 |
| **下载地址** | https://www.freedesktop.org/software/pulseaudio/releases/pulseaudio-17.0.tar.xz |
| **构建系统** | Meson |
| **Qt 特性** | FEATURE_pulseaudio |
| **依赖关系** | libsndfile |
| **输出文件** | libpulse.so, libpulse.so.0 |
| **头文件** | pulse/pulseaudio.h |
| **pkg-config** | libpulse.pc |

**配置选项**：
```bash
PULSEAUDIO_ENABLED="${PULSEAUDIO_ENABLED:-true}"
PULSEAUDIO_BUILTIN_VERSION="${PULSEAUDIO_BUILTIN_VERSION:-17.0}"
```

**交叉编译配置**：
```bash
--prefix=/usr
--cross-file=pulseaudio-cross-armhf.txt
--buildtype=release
-Dstrip=true
-Ddaemon=false           # 不编译 daemon
-Dclient=true            # 编译客户端库
-Dalsa=disabled          # 禁用 ALSA 后端
-Dx11=disabled           # 禁用 X11
-Ddbus=disabled          # 禁用 dbus
-Dsystemd=disabled       # 禁用 systemd
```

**精简配置**（嵌入式）：
- 禁用 daemon，只编译客户端库
- 禁用 X11、D-Bus、systemd 等桌面依赖
- 保留基础音频播放/录制功能

### FFmpeg - 多媒体编解码库

| 属性 | 值 |
|------|-----|
| **用途** | 音视频编解码、封装格式处理，Qt Multimedia 的核心依赖 |
| **当前版本** | 7.1 |
| **下载地址** | https://ffmpeg.org/releases/ffmpeg-7.1.tar.xz |
| **构建系统** | 自定义 configure |
| **Qt 特性** | FEATURE_ffmpeg |
| **依赖关系** | 无外部依赖 |
| **输出文件** | libavcodec.so, libavformat.so, libavutil.so, libswscale.so, libswresample.so |
| **头文件** | libavcodec/avcodec.h, libavformat/avformat.h 等 |
| **pkg-config** | libavcodec.pc, libavformat.pc, libavutil.pc 等 |

**配置选项**：
```bash
FFMPEG_ENABLED="${FFMPEG_ENABLED:-true}"
FFMPEG_BUILTIN_VERSION="${FFMPEG_BUILTIN_VERSION:-7.1}"
```

**交叉编译配置**：
```bash
--enable-cross-compile
--cross-prefix=arm-linux-gnueabihf-
--arch=arm
--target-os=linux
--prefix=/usr
--enable-gpl
--enable-version3
--enable-shared
--disable-static
--enable-small
```

**启用的编解码器**：
- **解码器**：H.264, H.265/HEVC, MPEG4, MJPEG, AAC, MP3, Opus, Vorbis, FLAC
- **编码器**：AAC, MP3, MJPEG, PNG, libopus
- **封装格式**：MP4, MOV, AVI, Matroska, OGG
- **网络协议**：HTTP, RTMP, TCP, UDP
- **滤镜**：scale, crop, overlay, resample

**ARM 优化**：
```bash
--enable-armv6
--enable-armv6t2
--enable-vfp
--enable-neon
--enable-fast-unaligned
```

### OpenSSL - 加密支持库

| 属性 | 值 |
|------|-----|
| **用途** | SSL/TLS 加密支持，Qt Network HTTPS 功能必需 |
| **当前版本** | 3.4.0 |
| **下载地址** | https://www.openssl.org/source/openssl-3.4.0.tar.gz |
| **构建系统** | Perl Configure |
| **Qt 特性** | FEATURE_ssl, FEATURE_openssl |
| **依赖关系** | 无外部依赖 |
| **输出文件** | libssl.so, libssl.so.3, libcrypto.so, libcrypto.so.3 |
| **头文件** | openssl/ssl.h, openssl/crypto.h |
| **pkg-config** | openssl.pc, libssl.pc, libcrypto.pc |

**配置选项**：
```bash
OPENSSL_ENABLED="${OPENSSL_ENABLED:-true}"
OPENSSL_BUILTIN_VERSION="${OPENSSL_BUILTIN_VERSION:-3.4.0}"
OPENSSL_TARGET="${OPENSSL_TARGET:-}"  # 自动检测
```

**目标平台映射**：
| 工具链前缀 | OpenSSL 目标 |
|-----------|-------------|
| arm-linux-gnueabihf | linux-armv4 |
| aarch64-linux-gnu | linux-aarch64 |
| x86_64-linux-gnu | linux-x86_64 |

**交叉编译配置**：
```bash
linux-armv4              # 目标平台
--prefix=/usr
--openssldir=/etc/ssl
shared                   # 动态库
no-tests                 # 禁用测试
no-docs                  # 禁用文档
no-apps                  # 禁用工具程序
no-asm                   # 禁用汇编优化
```

**SoC 特定优化**：
```bash
# i.MX6ULL (Cortex-A7)
CFLAGS="-march=armv7-a -mfpu=neon-vfpv4 -mfloat-abi=hard -O2"
```

## 版本更新指南

更新第三方库版本时，需要：

1. 修改 `config/third_party.conf` 中的版本号
2. 验证下载 URL 是否有效
3. 测试交叉编译是否成功
4. 检查 Qt 模块是否能正确检测新版本

## 添加新库

添加新的第三方库需要：

1. 在 `config/third_party.conf` 添加配置
2. 创建 `scripts/third_party/<lib>/builtin.sh` 脚本
3. 实现 `check`、`fetch`、`configure`、`build`、`install`、`clean` 函数
4. 将库添加到 `THIRD_PARTY_LIBS` 列表
5. 更新本文档
