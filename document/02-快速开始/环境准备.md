# 环境准备

在开始构建 Qt 之前，需要确保您的系统满足最低要求并安装所有必要的依赖。

## 系统要求

### 推荐配置

- **操作系统**: Ubuntu 20.04 LTS 或更高版本
- **CPU**: 4 核心及以上（推荐 8 核心以加快编译速度）
- **内存**: 8GB 及以上（推荐 16GB）
- **磁盘空间**: 至少 20GB 可用空间

### 支持的平台

- **Host Qt 构建**: Linux x86_64
- **Target Qt 交叉编译**: ARM (armhf, arm64), MIPS, x86 等（取决于工具链支持）

## 安装依赖

### 自动安装（推荐）

使用提供的脚本自动安装所有依赖：

```bash
sudo bash scripts/install-host-deps.sh
```

### 手动安装

如果您需要手动控制安装过程，可以执行以下命令：

#### 基础构建工具

```bash
sudo apt install -y build-essential cmake ninja-build perl ccache
```

#### OpenGL 和图形库

```bash
sudo apt install -y \
    libgl-dev libglvnd-dev libglx-dev libgles-dev libdrm-dev libegl1-mesa-dev
```

#### 字体和渲染库

```bash
sudo apt install -y \
    libfontconfig1-dev libfreetype6-dev libharfbuzz-dev
```

#### X11 和输入设备

```bash
sudo apt install -y \
    libx11-dev libxkbcommon-dev \
    libxcb1-dev libxcb-util-dev libxcb-xinerama0 libxcb-cursor0
```

#### 多媒体支持

```bash
sudo apt install -y \
    libgstreamer1.0-dev libgstreamer-plugins-base1.0-dev \
    gstreamer1.0-plugins-base gstreamer1.0-plugins-good
```

#### 音频和其他依赖

```bash
sudo apt install -y \
    libasound2-dev libpulse-dev libicu-dev libpcre2-dev libsqlite3-dev
```

## 验证工具链

安装完成后，验证关键工具是否正确安装：

### 检查编译工具

```bash
# GCC/G++
gcc --version
g++ --version

# CMake
cmake --version  # 应该 >= 3.22

# Ninja
ninja --version

# Perl
perl --version
```

### 检查开发库

```bash
# 检查 OpenGL
pkg-config --exists gl && echo "OpenGL found"

# 检查 X11
pkg-config --exists x11 && echo "X11 found"

# 检查 Freetype
pkg-config --exists freetype2 && echo "Freetype found"

# 检查 Fontconfig
pkg-config --exists fontconfig && echo "Fontconfig found"
```

### 检查 ccache（可选但推荐）

```bash
ccache --version
```

ccache 可以显著加快重复编译的速度，建议安装。

## 交叉编译环境

如果您需要构建 Target Qt（交叉编译），还需要准备交叉编译工具链：

### 安装交叉编译工具链

```bash
# ARM 32位
sudo apt install -y g++-arm-linux-gnueabihf

# ARM 64位
sudo apt install -y g++-aarch64-linux-gnu
```

### 验证交叉编译工具链

```bash
# ARM 32位
arm-linux-gnueabihf-gcc --version

# ARM 64位
aarch64-linux-gnu-gcc --version
```

## 常见问题

### Q: CMake 版本过低怎么办？

A: 安装更新的 CMake 版本：

```bash
# 使用 Kitware 官方源
wget -O - https://apt.kitware.com/keys/kitware-archive-latest.asc 2>/dev/null | gpg --dearmor - | sudo tee /usr/share/keyrings/kitware-archive-keyring.gpg >/dev/null
echo 'deb [signed-by=/usr/share/keyrings/kitware-archive-keyring.gpg] https://apt.kitware.com/ubuntu/ focal main' | sudo tee /etc/apt/sources.list.d/kitware.list >/dev/null
sudo apt update
sudo apt install cmake
```

### Q: Ninja 未安装怎么办？

A: 使用以下命令安装：

```bash
sudo apt install ninja-build
```

### Q: OpenGL 相关错误？

A: 确保安装了 OpenGL 开发包：

```bash
sudo apt install -y libgl-dev libglvnd-dev libglx-dev libgles-dev
```

### Q: 编译过程中内存不足？

A: 可以减少并行编译任务数：

```bash
PARALLEL_JOBS=2 bash build.sh
```
